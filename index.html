wtk-showcase by robap

This widget toolkit is for developers who use google closure and want the ability to style their ui widgets using jquery-ui css. This is not simply a google closure wrapper around jquery-ui. This is a separate ui widget library which takes advantage of jquery-ui css so that themes can be easily switched. You can try changing the theme for this demo by changing the select box in the upper right corner of this page.
